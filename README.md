
# ▣ The Headliner \ Modern Android News App

⧉  **The Headliner** brings together local & global news in a clean, distraction-free interface with smooth infinite scrolling, helping students and aspirants stay informed without information overload.

⧉ Trending Headlines| Categories(Business, Technology, Entertainment, Sports, Health, Science) | Search | Bookmarks | Offline Reading | Read History | Infinite Scrolling | Multi-Country News | Multi-Language Support | Notes |


## ⬤ Languages & Technologies
Kotlin | Jetpack Compose | MVVM | Hilt | Retrofit | OkHttp | Room | Coil | Coroutines | Flow | Navigation Compose | DataStore | Material 3 | REST API | Android SDK
